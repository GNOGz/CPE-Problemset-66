    เช่น (3) → 180, (10) → 300

Level 1

  1. MATH สร้างฟังก์ชั่นที่รับค่าจำนวนเต็ม 2 จำนวนแทนด้านของสามเหลี่ยมให้ return จำนวนเต็มที่แทนด้านของสามเหลี่ยมที่เหลือที่ยาวมากที่สุดที่จะเป็นไปได้

    เช่น (2,3) → 4, (19,43) → 61

  2. สร้างฟังก์ชั่นที่รับค่าจำนวนเต็มที่มากกว่า 1 และแสดงผลตั่งแต่เลข 1 ถึงเลขนั้นๆ แต่ถ้าเลขที่แสดงผลหาร 5 ลงตัวให้แสดง “X” แทนเลขนั้น

    เช่น (7) → 1234X67, (11) → 1234X6789X11

  3. สร้างฟังก์ชั่นที่รับค่าจำนวนเต็มและแสดงแม่สูตรคูณของเลขนั้นคูณ 1 จนถึงคูณ 12

    เช่น (9) → 9 18 27 36 45 ... 108, (20) → 20 40 60 80 ... 240

  4. สร้างฟังก์ชั่นที่รับค่าเป็น string ภาษาอังกฤษ 1 ตัวแล้ว return ค่าจำนวนสระภาษาอังกฤษที่อยู่ใน string นั้น

    เช่น (”frozen”) → 2, (”CPE”) → 1

  5. สร้างฟังก์ชั่นที่รับ string 1 ตัวแล้ว เช็คว่า string นั้นเป็น palindrome หรือไม่ โดย return เป็นค่าจริงหรือเท็จ (palindrome คือตัวที่อ่านจากหน้าไปหลังหรือหลังมาหน้าได้ตัวเดิม เช่น pop, mom, jkokj โดยไม่จำเป็นต้องมีความหมาย)

    เช่น (”tenet”) → true, (”cat”) → false

  6. MUST ให้รับจำนวนเต็ม 1 ตัวแล้วถ้าเลขนั้นหารเฉพาะ 3 ลงตัวให้แสดง “Fizz” ถ้าการเฉพาะ 5 ลงตัวให้แสดง “Buzz” ถ้าหารทั้ง 3 และ 5 ลงตัวให้แสดง “FizzBuzz” ถ้าไม่ลงตัวเลยให้แสดงเลขที่รับเข้าไปแทน

    เช่น (6) → 6, (10) → Buzz, (30) → FizzBuzz, (3) → Fizz

  7. MATH สร้างฟังก์ชั่นที่รับจำนวนเต็ม 2 จำนวน แทน อายุของ พ่อ(A) และ ลูก(B) แล้วแสดงผลบอกว่า อายุของพ่อจะเป็น 2 เท่าของลูกเมื่อผ่านไปกี่ปีหรือผ่านมาแล้วกี่ปี

    เช่น (42, 21) → ตอนนี้, (55, 30) → ผ่านมาแล้ว 5 ปี, (36, 7) → อีก 22 ปี (ไม่ต้องแสดงผลออกมาตามนี้ก็ได้ให้พอรู้ว่าเป็นเมื่อไหร่)

Level 2

  1. MATH สร้างฟังก์ชั่นที่รับจำนวนเต็มแล้ว return บอกว่าค่าที่รับมาใช่จำนวนเฉพาะหรือไม่ (จำนวนเฉพาะคือจำนวนที่มีแค่ 1 และตัวมันเองหารลงตัว)

    เช่น (7) → true, (15) → false, (191) → true

  2. สร้างฟังก์ชั่นที่รับ Binary String 1 ตัวเป็น input (Binary String คือ string ที่มีแค่เลข 0 กับ 1 เช่น “1001011”) แล้วแปลง Binary String นั้นให้เป็นเลขฐาน 10 แล้ว return ค่าออกมา

    เช่น (”1001”) → 9, (”11010010”) → 210
